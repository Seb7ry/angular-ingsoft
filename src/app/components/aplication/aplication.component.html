¿Qué herramientas usamos?


Plan

En esta parte se hizo uso de la herramienta Trello la cual nos permiten generar un ambiente de desarrollo colaborativo para asignar tareas y revisar pendientes. Esta herramienta es vital para que cualquier tipo de proyecto tenga una organización coherente y más aún cuando se trabaja con un gran volumen de personal que se puede dispersar en sus labores.


Code

Para el apartado de codificación se hizo uso del IDE Microsoft Visual Studio Code para crear la página mediante el framework para aplicaciones web Angular, durante el desarrollo se hace uso del controlador de versiones Git; esto, con el objetivo de llevar un orden en las modificaciones y actualizaciones hechas durante la codificación del proyecto.


Build

Debido a que este proyecto de Angular se trabaja de manera local se debe publicar en un repositorio para que cualquier usuario en la web pueda acceder a este, en nuestro caso se hizo uso de GitHub para postear toda la parte de codificación a esto, posterior en el servicio denominado Github Pages para que esto se pueda visualizar facilmente.
